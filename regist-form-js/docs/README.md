Project ini adalah program sederhana berbasis HTML, CSS, dan JavaScript untuk membuat form registrasi mahasiswa.

## Fitur

- Input nama mahasiswa
- Rekomendasi nama otomatis ketika user mengetik huruf tertentu
- Input NIM
- Dropdown mata kuliah
- Dropdown dosen berdasarkan mata kuliah yang dipilih
- Validasi form menggunakan JavaScript
- Menampilkan hasil registrasi setelah form berhasil disubmit
